About This Site!

This site was coded in Atom and generated using Metalsmith - a static site generator running on Node.js. Template engine used for layout and partials is Nunjucks. Styling is powered by LESS as CSS pre-processor, Responsive Grid Mixins from Bootstrap and LESS hat mixin library, with textured background from Subtle Patterns. The emblem is created in SVG and its animations are done using GreenSock's TweenLite & Friends. Scrolling mechanism is implemeted using TweenLite, jQuery and Bootstrap's ScrollSpy plugin, with some help of magical but still proposed Scroll Restoration feature of Chrome. Most JavaScript files are loaded at runtime with RequireJS as an AMD Loader to speed up initial loading. Fonts used in the site are Roboto, Roboto Slab and Lato, being loaded from Google Fonts and the icons used are from FontAweome!

Site is hosted on GitHub Pages mapped to domain purchased from name.com. It is sitting behind Cloudflare for performance improvement and security.
